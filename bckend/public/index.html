🚀 Blog API Server

Welcome to Blog REST API Backend

🌐 Base URL

Local: http://localhost:3000
Production: https://asks99.onrender.com
(Use the above base URL for all API requests) All Blogs: https://asks99.onrender.com/api/blogs

📚 Available Endpoints

GET /api/blogs

Get all blogs

GET /api/blogs/:id

Get single blog

POST /api/blogs

Create blog

PUT /api/blogs/:id

Update blog

DELETE /api/blogs/:id

Delete blog

PATCH /api/blogs/:id/like

Increase likes

PATCH /api/blogs/:id/view

Increase views

📝 Sample Request (Create Blog)

POST /api/blogs

{
"title":"My First Blog",
"content":"Hello World",
"author":"Karthikeyan"
}

✅ Sample Response

{
"_id":"65ab123",
"title":"My First Blog",
"content":"Hello World",
"likes":0,
"views":0,
"createdAt":"2026-03-05"
}
